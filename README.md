Welcome to the technical core of [famoussheamus.com](https://famoussheamus.com).

This repository is more than a portfolio; it is an **Engine for Generative Engine Optimization (GEO)**. In an era where AI agents and LLMs are the primary conduits of information, this codebase is architected to be semantically unambiguous, highly authoritative, and optimized for AI citation.

## 🚀 The Mission: Beyond SEO to GEO
As a **Fractional CTO** and **AI Architect**, I am building a "Digital Twin" of my professional knowledge. This site is designed to achieve:
*   **Semantic Grounding**: Every technical concept is mapped to verified [Wikidata](https://www.wikidata.org/) entities (Q-IDs) via JSON-LD.
*   **AI-First Readability**: MDX content structured specifically for LLM extraction and cross-referencing.
*   **High-Fidelity EEAT**: Hard-coded authority signals through `@graph` schema architecture to ensure AI search engines (Perplexity, Gemini, ChatGPT) can verify expertise and trust.

## 🛠️ Semantic Tech Stack
*   **Next.js 14.2+** (App Router) — High-performance core.
*   **JSON-LD @graph Architecture** — Centralized schema grounding in `src/lib/schema.ts`.
*   **Wikidata Integration** — 30+ verified entity mappings for topics like RAG, n8n, and Business Process Automation.
*   **Framer Motion** — Premium visual interactions for human engagement.
*   **Velite & MDX** — The logic layer for content processing.
*   **"No New Software" Philosophy** — An operational mandate to maximize existing stacks (n8n, Python) for ROI before adding complexity.
